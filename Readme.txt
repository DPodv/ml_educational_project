В ноутбуке представлен учебный проект "Обучение с учителем: качество модели" курса "Специалист по Data Science" Яндекс Практикума.

В проекте решается задача персонализации предложений и увеличения покупательской способности клиентов интернет-магазина "В один клик".
Для решения задачи мы обучили несколько моделей предсказывать уровень покупательской активности: 
- Дерево решений,
- Метод k-ближайших соседей,
- Метод логистической регрессии,
- Метод опорных векторов.
Для каждой из них мы подобрали гиперпараметры и выбрали наиболее точную модель. Изучив логику выбранной модели, мы выделили признаки, которые влияют на уровень покупательской активности. На основе этих признаков мы выделили сегменты покупателей и составили рекомендации для их привлечения к сервису.
